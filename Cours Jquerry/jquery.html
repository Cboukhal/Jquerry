Introduction:

Le JQUERY est une bibliotheque (library) de javascript libre et multiplateforme, créé pour faciliter l'écriture des scripts. Le JQUERY est un objet de javascript qui possede ses propres méthodes et propriétés, il nous permet de moins écrire pour plus coder.
Il contient les fonctionnalités suivantes:

  • Selection des elements de HTML
  • Manipulation des elements HTML
  • Manipulation CSS
  • Manipulation des evenements
  • Animations javascript
  • Parcours et modification du DOM
    • JQUERY est une library concise qui simplifie la navigation dans les documents HTML, la gestion des evenements, l'animation et les interactions... pour un développement web plus rapide.

    1. Présentation:

    John Resig est un développeur né le 8 mai 1984. Il est actuellement employé par la KHAN Academy en tant que responsable du developpement javascript. Il est notamment connu pour etre l'inventeur du JQUERY dont le but est de simplifier des commandes communes de javascript. John Resig etait un développeur qui travaillait sur ses propres projets et qui a constaté à quel point il était difficile de faire du code javascript compatible avec tous les navigateurs, donc il décida de créer sa propre bibliotheque JS pour remédier à ce probleme : le JQUERY est né.
    John Resig a mentionné pour la premiere fois une amélioration de la bibliotheque "Bihaviour" en Aout 2005.
    Le 16 Janvier 2006, il a officiellement sorti sa library sous le nom de JQUERY.

    3.1 Selecteur CSS

    le css est une méthode de Jquery qui renvoie une ou plusieurs propiétes de style pour les éléments sélectionnés.
    Exemple: voir jquery.html et script.js

    3.2 Selecteur hierarchique

    1. ascendant, descendant :
    syntaxe: $('parent enfant ...')
    2. enfant:
    syntaxe: $('pere > fils')
    3. element suivant:
    syntaxe: $('precedent + suivant')
    4. premier enfant:
    syntaxe: $('pere:first-child')
    5. dernier enfant:
    syntaxe: $('pere:last-child')
    6. premier element:
    syntaxe: $('pere:first')
    7. pas d'enfant:
    syntaxe: $('pere:empty')
    8. dernier element:
    syntaxe: $('pere:last')
    9. exclure un element:
    syntaxe: $('element:not')

    4. Les evenements

    Un evenement comme son nom l'indique est une action plus précisément qui survient à la suite d'un déclencheur spécifique.
    Ces actions vont provenir généralement du navigateur lui meme, soit des utilisateurs. Les evenements load et ready se déclenchent lorsque la page compléte est chargée tandis que les evenements comme le click, le dblclick, le mousemove... se déclenche lorsque l'utilisateur réalise l'action prédéfinie.

    4.1 L'evenement Ready

    En Jquery, la premiere methode qu'on va systématiquement appliquer est elle meme un evenement, c'est la methode ou l'évenement ready .
    syntaxe: $(document).ready(fonction(){ ensemble d'instructions. })

    4.2 Gestionnaire d'evenements

    Un gestionnaire d'evenement est une fonction destinée à être lancée en réponse à une action de l'utilisateur par exemple le click.
    Jquery fournit plusieurs manières de définir un gestionnaire d'evenement.
    syntaxe: $(selecteur).on('click', "", nom_de_la_fonction);
    $(selecteur).on('click', nom_de_la_fonction);
    $(selecteur).on('click', "", fonction(){instructions}); ou utiliser une fonction fléchée
    $(selecteur).click(fonction(){instructions}); ou utiliser une fonction fléchée

    4.3 les evenements de la souris

    la souris d'un ordinateur est un outil qui permet à l'utilisateur de réaliser de multiples actions comme cliquer, se déplacer, scroller ...

    Méthodes Actions
    click() lorsqu'on click sur la souris
    dblclick() lorsqu'on double click sur la souris
    hover() lorsqu'on survole un element
    mouseenter() lorsqu'on entre le pointeur de la souris dans un element
    mouseleave() lorsque le pointeur de la souris sort de l'élément
    mousedown() lorsqu'on presse un bouton de la souris
    mouseup() lorsqu'on relache un bouton de la souris
    scroll() lorsqu'on scrolle (utilisation de la molette)

    4.4 les evenements du clavier

    Le clavier comme la souris peuvent etre utilisés pour déclencher des instructions à la saisie d'une touche par l'utilisateur ou en relachant la touche.

    Méthodes Actions
    keydown() lorsqu'une touche du clavier est enfoncée
    keyup() lorsqu'on relache la touche du clavier
    keypress() lorsqu'on maintient une touche du clavier

    5. Les animations

    En mode natif, Jquery fournit un certains nombres de méthodes d'animations qui ajoutent un comportement de glissement et de décoration et fonctionnent sur la plupart des navigateurs.

    Les méthodes d'animations
    Méthodes Actions
    animate() execute une animation personnalisée d'un ensemble de propriété css
    delay() pour retarder l'execution des elements suivants dans la liste d'attente
    fadeIn() affiche les elements en les atténuant en opacité
    fadeOut() masque les elements en les atténuant en opacité
    fadeTo() ajuste l'opacité des elements
    fadeToggle() affiche et masque les elements en fonction de leurs opacités
    finish() arrete les animations en cours
    hide() masque les elements
    show() affiche les elements
    slideDown() affiche les elements avec un mouvement coulissant
    slideToggle() affiche ou masque les elements avec un mouvement coulissant
    slideUp() masque les elements avec un mouvement coulissant
    stop() arrete l'animation en cours
    toggle() affiche ou masque les elements
    eq() fonctionne avec l'indice d'un element liste ou tableau
